# 2022-Project
It is the embedded software of the design, which fixes the point of view of the camera with connected to a 
system that rotates around its own axis, collects data from the sensors of the system and transmits this 
data to the communication infrastructure at a distance.In addition, data can be transmitted to the main 
system by providing communication between two systems in free fall from a certain altitude. In other words, 
communication takes place between three different systems.
